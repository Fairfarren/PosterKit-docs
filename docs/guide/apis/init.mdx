# 初始化

## init()

`init`方法接受一个数组，类型为`CardData[]`

```tsx
kitBoxRef.init([] as CardData[])
```

### CardData

#### id

- Type: `string`
- Description: 卡片的唯一标识符，建议使用UUID。

#### width
- Type: `number`
- Description: 卡片的宽度，单位为像素。

#### height
- Type: `number`
- Description: 卡片的高度，单位为像素。

#### x
- Type: `number`
- Description: 卡片在`kitBox`中的x坐标，单位为像素

#### y
- Type: `number`
- Description: 卡片在`kitBox`中的y坐标，单位为像素

#### type
- Type: `'image' | 'text'`
- Description: 卡片的类型，分为图片和文字。

#### image
- Type: `HTMLImageElement`
- Description: 当`type`为`image`时，必填，图片对象。

#### text
- Type: `string`
- Description: 当`type`为`text`时，必填，文字内容。

#### fontSize
- Type: `number`
- Description: 当`type`为`text`时，必填，文字大小，单位为像素。

#### fontFamily
- Type: `string`
- Description: 当`type`为`text`时，可选，文字字体，可以是系统字体或自定义字体。

#### color
- Type: `string`
- Description: 当`type`为`text`时，可选，文字颜色，可以使用十六进制颜色值或RGB颜色值。

#### fontWeight
- Type: `string`
- Description: 当`type`为`text`时，可选，文字粗细，支持`normal`和`bold`。

#### fontStyle
- Type: `'normal' | 'italic'`
- Description: 当`type`为`text`时，可选，文字样式，支持`normal`和`italic`。

#### decoration
- Type: `'none' | 'underline' | 'line-through'`
- Description: 当`type`为`text`时，可选，文字装饰，支持`none`、`underline`和`line-through`。

## 示例

```tsx
<kitBox
  ref={kitBoxRef}
/>
```

```tsx
const image = new Image()
image.src =
  'https://material-center.meitudata.com/material/image/62fa22ac354485399.png'
image.crossOrigin = 'anonymous'

kitBoxRef.current.init([
    {
      id: new Date().getTime() + 1,
      width: 300,
      height: 200,
      x: 0,
      y: 0,
      text: '你好世界你好世界1234567890🤔abcdefghijklmnopqrstuvwxyz',
      type: 'text',
      fontSize: 32,
      fontFamily: 'cursive',
      color: '#db3f9178',
      fontWeight: 'bold',
      fontStyle: 'italic',
      decoration: 'line-through',
    },
    {
      id: new Date().getTime(),
      width: 300,
      height: 300,
      x: 0,
      y: 0,
      image,
      type: 'image',
    }
])
```

## 注意
如果`type`为`iamge`时，字段`image`必须是一个`Image`对象。
