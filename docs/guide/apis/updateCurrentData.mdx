# 更新卡片

## updateCurrentData

更新某个卡片需要id匹配

```tsx
kitBoxRef.updateCurrentData({} as CardData)
```

### [CardData](./type)

## 示例

```tsx
<kitBox
  ref={kitBoxRef}
/>
```

```tsx
kitBoxRef.current?.add({
  id: cardData.id,
  width: 300,
  height: 200,
  x: 0,
  y: 0,
  text: '你好世界你好世界1234567890🤔abcdefghijklmnopqrstuvwxyz',
  type: 'text',
  fontSize: 32,
  fontFamily: 'cursive',
  color: '#db3f9178',
  fontWeight: 'bold',
  fontStyle: 'italic',
  decoration: 'line-through'
})
```
